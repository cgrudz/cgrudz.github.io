Course information

Please note that there are no required textbooks and the lectures should be watched before class meeting sessions to prepare for activities. For deeper reading, recommended resources are the following:
  • Härdle, W.K. et al. Basic Elements of Computational Statistics. Springer International Publishing, 2017.
  • Rizzo, Maria L. Statistical computing with R. CRC Press, 2019.
  • Kutner, M. et al. Applied Linear Statistical Models 5th Edition / Applied Linear Regression Models 4th Edition
  • Review of statistical concepts, calculus and matrices.
  • R is required for the class. Students can get an installation file from the R project (free and open source software).
  • Students are required to install the editor RStudio Free Desktop Edition for some class assignments.
  • Students are also required to install LaTeX (free typesetting language). This is required to compile R Markdown, and is the standard for writing mathematical and statistical reports and publications.
  • This class will utilize DataCamp with free educational license. Instructions for accessing this license will be sent by the instructor.
  • Advanced students interested in the theory of programming are recommended Eloquent Javascript (free, creative commons).
