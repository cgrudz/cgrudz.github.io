Examination material

Finite probability space and discrete distributions

  • You should be able to compute a basic set theoretical probability question using some combination of unions and intersections.
  • You should be able to identify the parameters for either a binomial or a Poisson distribution from a word problem, and compute the probability of some range of possible observed outcomes. This should involve, e.g., the pbinom / ppois functions.

Continuous distributions

  • You should know how the normal, t, Chi-squared and F distributions are related to each other.
  • You should be able to compute the probability of observing some outcome in a range of possible values for any of the above, using the cdf.
  • You should be able to identify a theoretical quantile from any of the above using the inverse cdf.
  • You should understand some basic properties of the above distributions in terms of their symmetry and / or skewness.
  • You should be able to draw a random sample from any of the above distributions with a prespecified random seed.

Hypothesis testing and confidence intervals

  • You should be comfortable with running the t.test, varTest, and the shapiro.test and interpreting the results for the p-value. This includes understanding whether it is appropriate to use such a test due to the large sample size and/or the normality of the data, and when it is not.
  • You should know how to specify a different alternative hypothesis, e.g., for the t.test to run a one-sided test with its keyword arguments, and for a mean different than zero.
  • You will be asked to compute the p-value for some such test as above and possibly to interpret the result.
  • The ks.test will be included in the midterm and you should be comfortable subsetting some data across a binary variable and testing if the distributions for another variable are similar or not across the sub-populations.
