Course information

  • Faraway, J. Linear models with R, Second Edition. Chapman and Hall/CRC, 2016.
    • Free pre-print version from CRAN
    • Book web page
  • Kutner, M. et al. Applied Linear Statistical Models 5th Edition / Applied Linear Regression Models 4th Edition
  • Review of statistical concepts, calculus and matrices.
  • R is required for the class. Students can get an installation file from the R project (free and open source software).
  • Students are recommended to use the environment R-Studio (free integrated development environment).
  • Students are also recommended to install LaTeX (free typesetting language). This is required to compile R Markdown, and is the standard for writing mathematical and statistical reports and publications.
  • Free scientific programming lessons are available through Software Carpentry (nonprofit teaching programming for researchers).
  • Paul Hurtado's tutorials and R resources
  • Advanced students interested in the theory of programming are recommended Eloquent Javascript (free, creative commons).
