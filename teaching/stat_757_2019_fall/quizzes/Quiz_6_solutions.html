Problem 1:

Let \(\left\{Y_i\right\}_{i=1}^n\) be independent and identically distributed according to some distribution with mean \(\mu_Y\) and variance \(\sigma^2\), where \(\mu_Y\) is unknown, but we assume that \(\sigma^2\) is known. Let \[\begin{align} \gamma = P\left( -L < \frac{\overline{Y} - \mu_Y}{\sigma / \sqrt{n}} < L\right). \end{align}\] Identify which terms in the above equation are random and which are fixed, deterministic. Then, find a random subset of the real line for which there is a probability of \(100(1 - \gamma)\%\) that \(\mu_Y\) is contained in it.

Solution:

In the standard assumptions that we take in this course, all values except for \(\overline{Y}\) are fixed and deterministic â€“ \(\overline{Y}\) is a random variable which depends on the linear combination of the individual realizations of the samples.

We note that if, \[\begin{align} & \gamma = P\left( -L < \frac{\overline{Y} - \mu_Y}{\sigma/\sqrt{n}} < L \right)\\ \Leftrightarrow & \gamma = P\left( -L \frac{\sigma}{\sqrt{n}} < \overline{Y} - \mu_Y < L \frac{\sigma}{\sqrt{n}}\right) \\ \Leftrightarrow & \gamma = P\left( \overline{Y} - L \frac{\sigma}{\sqrt{n}} < \mu_Y < \overline{Y} + L \frac{\sigma}{\sqrt{n}}\right). \end{align}\] In this regard, we say that the complimentary probability of the above statement is that, \[ 1- \gamma = P\left( \mu_Y \in\left(-\infty, \overline{Y} - L \frac{\sigma}{\sqrt{n}}\right) \cup \left( \overline{Y} + L \frac{\sigma}{\sqrt{n}},+\infty\right)\right)\]

Problem 2:

Use the result of problem 1 in this problem. Suppose that we know \(\sigma=1\), \(L=1\), \(\gamma=.95\) and we have the observed values \(Y_1 = 1.5\), \(Y_2=2.5\), \(Y_3 = 2.75\), \(Y_4 = 5.25\). Someone says, there is a probability of \(95\%\) that the true mean lies between \(2.5\) and \(3.5\). Is this statement correct? Why or why not.

Solution:

We note that if the samples are given as above, \(\overline{Y}\) attains the value \(3\), \(\sqrt{n}=2\), and we have,

\[\begin{align} 2.5 &= \overline{Y} - L \frac{\sigma}{\sqrt{n}} \\ 3.5 &= \overline{Y} + L \frac{\sigma}{\sqrt{n}}. \end{align}\] However, if we treat the true mean as being a fixed, deterministic value, then the statement of whether \(\mu_Y\) lives within the interval of \(2.5\) to \(3.5\) is an exact question which is true or false. We only treat \(\overline{Y}\) as a random value until particular outcomes are given. This relates to the notion of confidence, in which we can say alternatively \(\mu_Y\) is contained in the interval \((2.5,3.5)\) with \(95\%\) confidence because we have used a procedure that would gaurantee this result based on \(95\%\) of the possible outcomes.
