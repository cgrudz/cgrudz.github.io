Problems:

Suppose we are considering a regression model for the gala data. We will define the following:

lmod1 <- lm(Species ~ Area + Elevation + Nearest + Scruz + Adjacent, data=gala)
lmod2 <- lm(Species ~ Area + Elevation, data=gala)
lmod3 <- lm(Species ~ Area + Elevation + Nearest + Scruz, data=gala)
lmod4 <- lm(Species ~ Elevation + Nearest + Scruz + Adjacent, data=gala)

Problem 1:

State the null and alternative hypothesis when comparing lmod1 and lmod2. What hypothesis test(s) can be used to determine if we reject the null?

Problem 2:

State the null and alternative hypothesis when comparing lmod1 and lmod3. What hypothesis test(s) can be used to determine if we reject the null?

Problem 3:

For each of lmod1, lmod2 and lmod3 answer if we can use a hypothesis test to compare this with lmod4. If so, state what the null and alternative hypothesis is and state what hypothesis test.
