Problem 1:

Suppose that we find that the predicted value for \(\beta_{area}\) is \(\hat{\beta}_{area}=-0.5\) and the confidence interval for \(\beta_{area}\) is given as \(\left(-1.5, .5\right)\). What can you comment about the uncertainty in this parameter?

Solution:

We should note that the confidence interval contains the value zero. Therefore, dual to the hypothesis test, we cannot rule out the possibility of the parameter being equal to zero with \(5\%\) significance.

Problem 2:

Suppose that the predicted value for \(\beta_{adjacent}\) is \(\hat{\beta}_{adjacent} =-50\) and that the confidence interval for \(\beta_{adjacent}\) is given as \((-95, -5)\). What can you comment about the uncertainty in this parameter?

Solution:

Here we note that zero is not contained within the confidence interval. However, we see that relative to the scale of the response, the range of possible values that the parameter might attain is extremely wide. A one unit increase in the value adjacent can range from a value in the first quartile of the response to the third quartile. The effect of this parameter while likely negative, has a scale that is very uncertain (assuming a one unit increase is in the scope).

Problem 3:

Given the above confidence intervals, what can you comment about the hypothesis test, \[\begin{align} H_0&: \beta_{area}=\beta_{adjacent} = 0\\ H_1&: \boldsymbol{\beta} \neq 0? \end{align}\]

Solution:

We should note that the univariate confidence intervals, like the univariate t-tests, do not tell us information about the simultaneous hypothesis test. Rather, like the joint confidence region, this needs to be tested with an F-test over both parameters simultaneously.
